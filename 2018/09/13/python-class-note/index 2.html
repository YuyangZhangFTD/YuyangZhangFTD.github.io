抽象方法

__new__和__init__

__init__() 方法用于给属性赋值，不可以有返回值，__new__()多用于构造不可变对象，必须有返回值，返回实例对象。构建新实例的时候，先调用__new__()初始化一个实例，然后调用__init__() 进行属性赋值。

super

当子类重写父类构造方法时，为了避免父类属性没有初始化，有两种方式，第一种直接调用父类方法

也可以直接用super()方法

self和cls

实例方法参数需要带self，而类方法需要带cls。
