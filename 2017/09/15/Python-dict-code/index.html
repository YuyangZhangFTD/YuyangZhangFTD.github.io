PyDictObject

其中PyObject_HEAD 用于表明首地址，ma_fill 为活动槽以及哑槽的综述，但已有的记录被删除后，该记录的位置被标记为哑槽，ma_used 为活动槽总数，ma_mask 为数组长度减一的标记，用于计算槽的索引，ma_table 为数组本身，ma_smalltable为长度为 8 的初始数组，即字典的初始大小。

  3. 返回对象

分配一个字典结构，内部表的尺寸为8。

        • slot 0 is not used so return it
